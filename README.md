# HTML Download Test

A [simple webpage](index.html) to test downloads. By default, it points to the [lorem-ipsum.pdf](lorem-ipsum.pdf) file included. This can be changed by setting the `data_url` variable.

This test webpage was motivated by certain mobile browsers not behaving properly with blob downloads. In particular, PDF documents from a blob URL cannot be saved to the phone.

Access it live [here](https://isaacchua.github.io/html-download).
